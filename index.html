What is this fr.ffontenoy.e4.stickyview for?

This plugin is used for easily creating a sticky view for a pure e4 application. A sticky view is a view that can be displayed on every defined perspective and that remains displayed when switching to another perspective. This can be used for example for implementing a help window (next coming project).

How to use it?

In order to defined your own sticky view, you just need to extend the extension point stickyviewExtensionPoint giving the following attributes:

  • partId: The id that will be given to the created part. This id will also be used as a basis for the open/close command and handler.
  • containerData: Usual container data. Used in particular for defining the opening size
  • position: Defines the opening position. Should be one of left, right, top or bottom;
  • partClass: Defines the class for the part implementation. This should be a usual part implementation for E4 (with a @PostConstruct)
  • label: Defines the label of the part (used for the tab label).
  • visibleAtStartup: Defines whether the sticky view should be displayed at startup or not. Should be true or false.
  • shortcut: The shortcut sequence for opening and closing the sticky view (e.g.: M1+M2+H)
